  2. CSS

    Primary Use Cases Key Features/Benefits
    CSS is used to style web pages
    • Relatively simple language to learn
    • Very powerful that can be used to create very complex and sophisticated styles
    • CSS is very flexible

  3. JavaScript

    Primary Use Cases Key Features/Benefits
    JavaScript is used to add interactivity to web pages
    • It helps create interactive web pages that respond to user input
    • create games and animations that are both fun and visually appealing
    • used to create web applications that are both powerful and user-friendly

  4. MongoDB

    Primary Use Cases Key Features/Benefits
    MongoDB is a document-oriented database
    • it can be used to store a wide variety of data
    • it can be easily scaled up or down to meet the needs of your application
    • it can handle large amounts of data quickly and easily

  5. React js

    Primary Use Cases Key Features/Benefits
    React is a JavaScript library for building user interfaces
    • React is declarative
    • it can render UIs quickly and smoothly
    • it can be used to build a wide variety of UIs

  6. Express js

    Primary Use Cases Key Features/Benefits
    Express.js is used to build web applications and APIs
    • it is very fast
    • it provides a powerful routing system
    • it is easy to use

  7. Node js

    Primary use case Key Features/Benefits
    Node.js is used to build scalable and performant web applications and APIs
    • it is very scalable, and it can be used to build applications that can handle millions of concurrent requests
    • it is event-driven, which makes it very efficient at handling concurrent requests
